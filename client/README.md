# Client (React + Vite)

This folder contains the React + TypeScript frontend for the Personal Finance Tracker.

## Setup

```bash
npm install
```

## Environment

Optional `client/.env`:

```bash
VITE_API_URL="http://localhost:5000"
```

If not set, the client defaults to `http://localhost:5000`.

## Development

```bash
npm run dev
```

## Production

```bash
npm run build
npm run preview
```
